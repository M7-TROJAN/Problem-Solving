Description
-----------
The JPG File Verification project is a simple command-line program that checks whether a given file is a valid JPG image file. It uses a basic validation technique to determine if the file's first three bytes match the signature of a JPG file. The project is implemented in the C programming language.

===============================================================================

Getting Started
---------------
To run the JPG File Verification project on your local machine, follow these steps:

1. Compile the source code using a C compiler.
2. Run the compiled executable file.
3. Provide the file name as a command-line argument when running the program.
   Example: ./jpg-verification filename.jpg
4. The program will output whether the file is possibly a JPG file or not.
5. Additionally, it will display the hexadecimal values of the first three bytes for verification purposes.

Note: The project is written in C and requires a C compiler to compile and run the code.

===============================================================================

Project Structure

- main.c: Contains the main program logic to verify a file as a JPG image.
